# 仿海底捞订座排队小程序/H5/app

依托于“api工厂”提供的各种api接口以及后台管理功能，使用uni-app框架，开发该小程序应用。

本应用为模仿海底捞的订座排队小程序的功能，欢迎大家一起来贡献代码～
